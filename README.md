Bem-vindo ao repositório **Store.API**! Este projeto serve como um estudo pessoal da construção de uma Web API simples utilizando tecnologias de ponta como .NET, Entity Framework e C#. 

## Entidades Principais

### Produto

- **Descrição**: Representa um produto em estoque.
- **Atributos**: ID, Nome, Preço de Venda, Preço de Compra, Estoque, Categoria, Data de Criação e Atualização.

### Categoria

- **Descrição**: Categoriza produtos para melhor organização.
- **Atributos**: ID, Nome.

### Venda

- **Descrição**: Registra uma venda de produtos.
- **Atributos**: ID, Preço, Data de Começo e Fim.

### Item da Venda

- **Descrição**: Detalhes dos produtos vendidos em uma transação.
- **Atributos**: ID, Produto, Quantidade, Preço, Venda.

## Instalação

Para explorar este projeto localmente, siga estas etapas:

1. Clone o repositório: `git clone https://github.com/your-username/Store.API.git`
2. Navegue até o diretório do projeto: `cd Store.API`
3. Abra a solução no Visual Studio ou no seu IDE preferido.
4. Restaure as dependências usando o NuGet.
5. Compile e execute a aplicação.
